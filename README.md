# Trabajo Final

Este repositorio contiene el enunciado, los materiales y el desarrollo del **Trabajo Final (Tarea 2)** para la asignatura **Análisis Inteligente de Datos**.

### Descripción del Proyecto
El objetivo del trabajo es evidenciar las cuatro etapas del análisis:
**Selección, Limpieza, Exploración y Comunicación**
Se realiza un análisis exploratorio sobre el dataset **Mental Health and Social Media Balance**, compuesto por 500 registros individuales, para comprender cómo los hábitos digitales (tiempo de pantalla, uso de redes sociales) y factores de estilo de vida (sueño, ejercicio), entre otros, influyen en el bienestar emocional.

### Estructura del Repositorio
- `index.qmd`: Archivo fuente en Quarto con todo el código y análisis.
- `index.html`: Reporte final generado (Salida).
- `Mental_Health_and_Social_Media_Balance_Dataset.csv`: Conjunto de datos utilizado.
- `README.md`: Información general del proyecto.

### Datos
- **Nombre:** Mental Health and Social Media Balance Dataset
- **Fuente:** [Kaggle](https://www.kaggle.com/datasets/ayeshaimran123/social-media-and-mental-health-balance)
- **Registros:** 500 filas individuales.
